Background Images for Climatology Website

Place your background images in this folder and reference them in the website.

Recommended image names and sizes:

1. hero-cityscape.jpg (1920x1080px) - City skyline at sunset for hero section
2. mission-sky.jpg (1920x1080px) - Beautiful sky/clouds for mission section  
3. mountain-range.jpg (800x600px) - Snow-covered mountains for feature section
4. city-sunset.jpg (800x600px) - City sunset for CTA section
5. einstein-quote-bg.jpg (1920x1080px) - Cityscape for quote section

To use these images:
1. Add your images to this folder
2. Open script.js and update the backgroundConfig object
3. Or use the JavaScript functions in the browser console

Example:
ClimatologyWebsite.setHeroBackground('images/hero-cityscape.jpg');
